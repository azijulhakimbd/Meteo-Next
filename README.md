# 🌦️ Weather Pro

A modern, responsive and smart weather dashboard built with React, Tailwind CSS, and OpenWeather/Open-Meteo API.  
It provides real-time weather updates, hourly temperature charts, and 7-day forecasts with a beautiful UI.

---

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Get weather from current location
- 🌡️ Real-time temperature display
- 💧 Humidity, rain, wind & pressure info
- 📊 Hourly temperature chart (Recharts)
- 📅 7-day weather forecast
- 🌙 Dark / Light mode support
- 📱 Fully responsive (Mobile, Tablet, Desktop)

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Recharts
- Open-Meteo API / Weather API
- Lucide Icons
